### 内存大小
```
1 byte = 4 bits
1 kb = 1024 bytes = 256 int = 128 long long
1 mb = 1024 kb = 2.6 * e5 int = 1.3* e5 longlong 
1 Gb = 1024 mb = 2.7 * e8 int = 1.3 8 e8 longlong
```
